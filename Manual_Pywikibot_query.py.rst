 **query.py** is a library for
`pywikipediabot <Manual:Pywikipediabot>`__ used by
`casechecker.py <Manual:Pywikipediabot/casechecker.py>`__ and
`copyright\_clean.py <Manual:Pywikipediabot/copyright_clean.py>`__.
